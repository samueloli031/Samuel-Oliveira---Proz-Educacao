**🐾 Sistema de Gerenciamento de Petshop**

Este é um projeto desenvolvido em **Python** focado em aplicar conceitos fundamentais de lógica de programação, estruturas de dados e automação de processos para o cenário real de um petshop.
O sistema foi desenhado para simplificar o fluxo de atendimento, organizando o cadastro de clientes, pets e o controle de serviços de forma rápida e eficiente.

Funcionalidades:
Cadastro Estruturado: Registro de clientes e seus respectivos pets utilizando coleções dinâmicas (dicionários e tuplas) para garantir a integridade dos dados.
Controle de Serviços: Menu interativo para seleção de banho, tosa e consultas.
Fluxo Automatizado: Cálculo automático de ordens de serviço e geração de resumos de atendimento.
*Interface via Terminal: Menu limpo e intuitivo para navegação do usuário.

Tecnologias Utilizadas:
Python 3.x (Lógica central, loops de repetição, funções e manipulação de dicionários/tuplas)
